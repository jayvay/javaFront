정규식(Regular Expression) 연습

문자열 : 

        자바 : ^[~~~]$
        자바스크립트 : 
            형식 : /~~~/플러그인 , /[~~~]/플러그인   
                플러그인 :
                g(global) : 전체 범위에서의 처리
                m(multi line) : 여러 줄에 걸쳐 처리(textArea 태그)
                s(single line) : 한 줄에서 처리 (기본값으로 생략한다) 
                i : 영문자인 경우에 대/소문자 구별을 하지 않는다
                u(unicode) : 유니코드 구별 처리할 때 사용

            정규식을 실행하기 위해 사용하는 함수
            1. 사용자입력 문자열.match(정규표현식 or 정규표현식변수) : 포함 유무에 대한 비교할 때 사용하면 편리함
            2. 정규표현식변수.test(사용자입력 문자열) : 구체적인 문자(열) 간의 비교할 때 사용하면 편리함

Front에서 서버로 값을 전송(form태그 사용시)할 때는 사용자 값 입력 후 반드시 유효성 검사를 거쳐야 한다.
